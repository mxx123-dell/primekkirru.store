## 🚀 Cách deploy:
1. Fork repo này.
2. Vào [Render.com](https://render.com) → New Web Service → Connect GitHub.
3. Chọn repo này, ngôn ngữ PHP.
4. Khi chạy lần đầu, Render sẽ tự động tải & giải nén full source từ Drive.
